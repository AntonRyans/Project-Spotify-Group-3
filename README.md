# Project-Spotify-Group-3

#Setup Instructions for Project-Spotify-Group-3

1. To set up the project environment, make sure you have **Python 3.13** installed.
2. Clone the repository and enter the project folder.
3. Create a virtual environment.
4. Activate the virtual environment.
5. Install all the required packages from requirements.txt
6. Verify your installation using _pip list_.
